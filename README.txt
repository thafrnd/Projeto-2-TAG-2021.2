Para este projeto é considerada uma universidade oferece anualmente uma lista de
projetos financiados e abertos a participação de alunos. Cada projeto possui quantidades mínima e máxima de alunos que podem ser aceitos em determinado projeto, bem como requisitos  que os alunos devem possuir para serem aceitos. Esses requisitos de histórico e tempo são pré-avaliados e cada aluno. Este projeto pede a elaboração, implementação e testes com a solução final de emparelhamento máximo estável para os dados fornecidos. Na aplicação da variação do algoritmo Gale-Shapley adotada todo aluno começa livre e todos os projetos começam vazios, dessa forma vamos iterar sobre os projetos livres e para cada projeto vamos iterar todos os alunos, se tiver livre adiciona ao projeto e se não estiver livre mas preferir esse projeto ele é retirado do projeto, dessa forma o antigo projeto fica livre, caso o projeto não esteja livre passa para o próximo da fila, as iterações são registradas com auxilio de uma lista que armazena 10 iterações e a faixa delas é selecionada manualmente.
